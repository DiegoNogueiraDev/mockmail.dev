# Watch Dashboard - Estrutura

## Tech Stack
Next.js 15, React 19, Tailwind v4, Recharts, Lucide Icons

## API Routes (app/api/)
- chart-data/route.ts - Dados de gráficos
- daily-stats/route.ts - Estatísticas diárias
- email-boxes/route.ts - Caixas de email
- errors/route.ts - Logs de erros
- health/route.ts - Health check
- metrics/route.ts - Métricas

## Comandos
- `npm run dev` - Dev com Turbopack
- `npm run build` - Build produção
